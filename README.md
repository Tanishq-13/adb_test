## Work Completed

* Implemented React UI using functional components + hooks only.

* Backend uses MongoDB via PyMongo (no Django models/serializers/SQLite).

* Added helper functions for DB operations and clean error handling.

* Implemented:

GET /todos/ → fetch todos
POST /todos/ → create todo

* Frontend refreshes todo list on every successful create.

* Fully followed the Docker setup (app, api, mongo containers).
